## 📖 About The Project

The **Geo Weather App** is a responsive web application designed to deliver accurate and real-time weather updates based on the user's current location or searched city.

This project combines modern frontend technologies with weather APIs to create a fast, clean, and interactive user experience.

---

## ✨ Features

- 📍 Automatic geolocation weather detection
- 🔎 Search weather by city name
- 🌡️ Live temperature updates
- ☁️ Weather condition indicators
- 💨 Wind speed and humidity data
- 🌅 Sunrise and sunset information
- 📱 Fully responsive design
- ⚡ Fast-loading GitHub Pages deployment

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/otieno272/geo-weather.git
```

### Navigate into the project folder

```bash
cd geo-weather
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

---

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

### Build the application

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
REACT_APP_WEATHER_API_KEY=your_api_key_here
```
